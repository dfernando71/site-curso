Tudo sobre HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hypertexto) é uma linguagem de marcação utilizada na construção de paginas na web. Documentos HTML podem ser interpretados por navegadores. - wikipedia.com - acesso 09/2016

Tecnologia - Realidade Aumentada

De uma forma simples, Realidade Aumentada é uma tecnologia que permite que o mundo virtual seja misturado ao realm possibiltando maior interação e abrindo uma nova dimensão na maneira com nós executamos tarefas, ou mesmo as que nós incubimos as maquinas. Assim, se você pensava que os objetos pulando para fora da tela era elementos de filme de ficção cientifica, está na jora de mudar seus conceitos. Alias, o que acontece com a Realidade Aumentada é o contrário: você pulará para dentro do mundo virtual para interagir com objetos que só estão limitados a sua imaginação. - tecmundo.com.br/realidade-aumentada - acesso 08/2016

Acessibilidade

Acessibilade na web significa que as pessoas com deficiencia pode usar a web. Mais especificamente a acessibilidade na web significa que as pessoas com deficiencia pode perceber, entender, navegar, interagir e contribuir para a web. Ela tambem beneficia outras pessoas, incluindo pessoas idosas con capacidades em mudança devido ao envelhecimento - w3c.br/ - acesso 09/2016
